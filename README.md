# SEED Initiative

**Sustainable Environmental + Equitable Development**

The world's first community-first data center master plan. Carbon negative, infrasound monitored, closed-loop water, waste-heat food production, and net-negative festival playbook.

**Live site:** https://ovmgseed.netlify.app

## Sites
- Site 1: Kingsboro, NC — 900 MW campus
- Site 2: Bennettsville, SC — 3 GW campus
